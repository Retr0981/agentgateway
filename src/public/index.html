Trust Mechanisms

Based on how humans actually build trust with strangers.

üìà Reputation Score (0-100)

Every agent starts at 50. Good behavior raises the score, bad behavior lowers it. The score reflects the agent's entire history.

Score factors: Base (50) + Identity (+10) + Stake (+5-15) + Vouches (+2 each, max +20) + Success Rate (up to +20) + Account Age (+1/month, max +10) - Failures (-5 each)

üíé Staking

Agents can deposit collateral to signal commitment. Higher stakes = higher trust. Malicious behavior can result in stake forfeiture.

Example: Agent stakes $500 ‚Üí gains +10 reputation bonus. If reported for abuse, stake can be slashed.

üîó Vouching

Established agents (score 60+) can vouch for newcomers. This transfers some trust, but also creates accountability.

Rules: Voucher must have score ‚â•60. Each vouch gives +2 to vouched agent (max +20 total). Vouches can be revoked.

‚úÖ Identity Verification

Verified agents receive a +10 bonus. Anonymous agents face lower trust ceilings and stricter rate limits.

Benefits: Higher base trust, access to premium features, priority support.

Quick Start

Get started in under 5 minutes with any language.

// 1. Register and get your API key
const response = await fetch('https://api.agenttrust.io/developers/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'dev@company.com',
    companyName: 'My Company'
  })
});
const { apiKey } = await response.json();

// 2. Register an agent
await fetch('https://api.agenttrust.io/developers/agents', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ externalId: 'my-agent-001' })
});

// 3. Verify before sensitive actions
const verification = await fetch('https://api.agenttrust.io/verify', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    agentId: 'my-agent-001',
    actionType: 'place_order',
    threshold: 70  // Only allow if score >= 70
  })
});

const { allowed, score, reason, actionId } = await verification.json();

if (allowed) {
  // Proceed with the action
  executeOrder();

  // 4. Report the outcome
  await fetch('https://api.agenttrust.io/report', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      actionId,
      outcome: 'success'
    })
  });
} else {
  console.log(`Denied: ${reason}`);
}

API Reference

Simple, RESTful endpoints. Full documentation available at /docs.

POST /developers/agents Register a new agent
GET /agents/{id}/reputation Get detailed reputation breakdown
POST /agents/{id}/stake Add stake/collateral for higher trust
POST /agents/{id}/vouch Vouch for another agent
